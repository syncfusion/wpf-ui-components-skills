# Syncfusion® WPF UI Components Skills

**Agent Skills** for building accurate, production-ready applications with Syncfusion WPF components. Install these skills into your IDE (VS Code, Cursor, CodeStudio) to enable AI assistants to generate correct Syncfusion WPF code grounded in official APIs, patterns, and best practices.



## Key Features

- **Comprehensive Component Coverage** — Skills for Syncfusion WPF UI components with detailed implementation patterns
-  **Accurate APIs** — Official component props, events, and methods  
- **Production-Ready Code** — Official APIs, props, events, and methods with proven best practices
-  **Best Practices** — Proven patterns for state management and feature integration  
- **Multi-Purpose Skills** — Both common/utility skills and component-specific skills for flexible AI-assisted development
-  **Theming & Styling** — Supports Windows 11 light, Windows 11 dark, Fluent light, Fluent dark, system theme etc.,
- **Built-in Theme Knowledge** — Syncfusion theme configurations and styling capabilities
-  **Globalization** — RTL and accessibility guidance  
- **Best Practices** — State management patterns, feature integration, and globalization (RTL)

- **Rich References** — Implementation guides and code examples for common use cases and advanced features



## Quick Start

### Install Skills

```bash
npx skills add syncfusion/wpf-ui-components-skills -y
```

Your AI assistant will automatically load these skills in supported IDEs.

### Explore Skills

Each skill is self-contained:
- **SKILL.md** — Core concepts and patterns
- **references/** — Detailed implementation guides by feature area



## How AI Assistants Use These Skills

With skills installed, your AI assistant:
1. **Loads** relevant skills based on your query (e.g., "Add a Syncfusion DataGrid with paging, sorting, and filtering." → `syncfusion-wpf-datagrid`)
2. **Extracts** accurate APIs, props, and proven patterns
3. **Generates** production-ready code with proper imports and types
4. **Enforces** best practices (theming, accessibility, performance)
