# Appearance and Customization (summary)

This file covers the core appearance and customization topics for the CheckListBox. Detailed examples and extended templates have been moved to a companion file to keep this reference concise.

Quick pointers:
- Color and state brushes: `Foreground`, `Background`, `MouseOverBackground`, `SelectedItemBackground`
- Checkbox position: `CheckBoxPlacement` (or `CheckBoxAlignment` in some versions)
- RTL and mirroring: use `FlowDirection` on the control or parent `Window`
- Templates and styles: prefer `ItemTemplate` and `ItemContainerStyle` for item-level customization; use `ControlTemplate` for full control overrides
- Themes: use `SfSkinManager` or resource dictionaries to apply Syncfusion themes

For extended examples, code snippets, and troubleshooting see: [appearance-customization-examples.md](appearance-customization-examples.md)
