# Step Interval in WPF Double TextBox

The `DoubleTextBox` control allows you to increase or decrease the value by pressing up-arrow and down-arrow keys or using the mouse wheel. The `ScrollInterval` property specifies the increment/decrement. Default is `1`.

## Change Value on Up/Down arrow key
Set `ScrollInterval` to control the amount changed on arrow keys.

## Change Value on Mouse Wheel
Enable `IsScrollingOnCircle` to allow mouse wheel scrolling.

## Change Value on Click and Drag
Use `EnableExtendedScrolling` to allow click-and-drag value changes.

## Allow or restrict selection on focus
Use `TextSelectionOnFocus` to control automatic selection when focused.
