# Changing Double Value in WPF Double TextBox

The `DoubleTextBox` allows the user to change the value using the `Value` property. You can set `PasteMode=Advanced` to customize paste behavior, enable `ShowSpinButton` to show up/down buttons, handle `ValueChanged` events, and use `NullValue`/`UseNullOption` for null handling. Watermark and watermark templates are supported.
