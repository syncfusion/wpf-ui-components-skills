# Restriction or Validation in WPF Double TextBox

How to validate or restrict the `DoubleTextBox` control value.

## Min/Max limits and validation modes
Use `MinValue` and `MaxValue`. Choose validation timing with `MinValidation`/`MaxValidation` (`OnKeyPress` or `OnLostFocus`).

## MaxValueOnExceedMaxDigit / MinValueOnExceedMinDigit
Control behavior when input exceeds digit limits.

## Restrict number of decimal digits
Use `NumberDecimalDigits`, `MinimumNumberDecimalDigits`, `MaximumNumberDecimalDigits`.

## Read only mode
Set `IsReadOnly` and optionally `IsReadOnlyCaretVisible`.
