# Appearance in WPF DoubleTextBox

This section deals with the appearance of `DoubleTextBox` control and contains the following topics.

## Setting the Foreground

The `DoubleTextBox` control `Foreground` can be modified based on the value of the control (positive, negative, zero).

### Foreground for Positive Value
Set `PositiveForeground` to change color when `Value` is positive.

### Foreground for Negative Value
Set `NegativeForeground` and enable `ApplyNegativeForeground`.

### Foreground for Zero Value
Set `ZeroColor` and enable `ApplyZeroColor`.

## Setting the Background
Use `Background` to change control background.

## Setting the Corner Radius
Use `CornerRadius` to round the control border.

## Apply Background for Selection
Use `SelectionBrush` and `SelectionOpacity` to style selected text.

## Align Value
Use `TextAlignment` to align the displayed value.

## Setting ToolTip
Set `ToolTip` to show hover information.
