# Visual Structure

The ChromelessWindow is composed of several visual parts that can be templated and styled:

- `PART_TitleBar` - the title bar host
- `PART_Icon` - the window icon element
- `TitlePresenter` - displays the title text
- Title buttons - close/minimize/maximize/restore
- ContentPresenter - hosts the window content

Understanding these parts helps when overriding templates or adding controls.
