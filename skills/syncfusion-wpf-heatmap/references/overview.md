# Overview

`SfHeatMap` visualizes tabular data values as gradient colors instead of raw numbers. Use it to highlight ranges and patterns across rows and columns.

Key features:
- TableMapping and CellMapping
- Color mapping with `ColorMappingCollection`
- Legend via `SfHeatMapLegend`
- Virtualization for large datasets

When to choose HeatMap:
- When the user needs a quick visual overview of dataset concentrations and ranges
- For dashboards and compact visual summaries

See: [references/getting-started.md](references/getting-started.md) for examples.
