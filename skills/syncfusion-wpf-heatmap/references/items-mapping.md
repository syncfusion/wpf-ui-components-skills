# Items Mapping

`SfHeatMap` supports two mapping styles:

- `TableMapping`: each item represents a row with multiple numeric columns (good for year columns)
- `CellMapping`: each item represents a single cell (row, column, value)

Choose `TableMapping` when your model has properties for each column. Choose `CellMapping` when data is normalized into row/column/value triplets.

Examples and XAML snippets mirror those in the main docs.
