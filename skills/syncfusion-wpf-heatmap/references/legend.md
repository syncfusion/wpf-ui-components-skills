# Legend

Use `SfHeatMapLegend` to summarize color ranges. Pass the same `ColorMappingCollection` used by the heatmap.

Modes:
- `Gradient` — continuous gradient bar
- `List` — labeled buckets

Orientation: `Horizontal` or `Vertical` — choose based on layout.

Example:

```xaml
<syncfusion:SfHeatMapLegend LegendMode="Gradient" ColorMappingCollection="{StaticResource colorMapping}" />
```
